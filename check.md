# 노선 파일 무결성 검사

모든 '역 파일' 의 모든 '종착역' 이 'roads.txt'에 존재하는지를 검사해야 함.
roads.txt의 그 종착역 기준 역이 존재하는지도 검사해야 함

```
roads.txt 열기
종착역을 연결리스트로 저장
 / 종착역 내에서 역을 연결리스트로 저장

역 파일에 대해...
  파싱한 노선(unique) 마다...
    '노선의 종착역' 이 roads.txt에 있는지 확인
    없으면 오류 #1 출력하고 다음 노선 진행
    '종착역 노선' 에 '역 이름' 이 있는지 확인
    없으면 오류 #2 출력
    
```

## 오류

오류 #1: 노선 '$1' 가 없습니다. ($2.txt)
오류 #2: 노선 '$1'에 $2가 없습니다.
